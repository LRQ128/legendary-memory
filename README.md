# 日常账本 📒

一款简洁、美观的个人记账应用，基于 Flutter 开发。

## 功能特性

- 📊 **首页概览** — 月度收支汇总、预算进度、最近记账
- 📅 **日历视图** — 按日查看账单，点击日期查看详情
- 💰 **资产管理** — 多账户管理、债务追踪、理财统计
- 📈 **数据统计** — 折线图/饼图展示收支趋势和分类占比
- 👤 **个人中心** — 自动记账、周期记账、标签管理、数据备份
- 🌈 **清新设计** — 绿色主题，轻量简洁，交互流畅

## 技术栈

- **Flutter** 3.x + Dart 3.x
- **SQLite** 本地数据存储
- **Provider** 状态管理
- **fl_chart** 图表渲染

## 开始使用

```bash
# 克隆仓库
git clone https://gitee.com/li-ruiquan-123/personalized-customization.git

# 进入目录
cd personalized-customization

# 安装依赖
flutter pub get

# 运行
flutter run
```
